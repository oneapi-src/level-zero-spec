Device-ECC properties#

By default, {s}DeviceGetEccState returns the current state of the ECC counters.

Additionally to get default ECC state, pass zes_device_ecc_default_properties_ext_t to {s}DeviceGetEccState as pNext member of zes_device_ecc_properties_t.

zes_device_ecc_properties_t getState = {ZES_STRUCTURE_TYPE_DEVICE_ECC_PROPERTIES};
zes_device_ecc_default_properties_ext_t ext = {ZES_STRUCTURE_TYPE_DEVICE_ECC_DEFAULT_PROPERTIES_EXT};
getState.pNext = &ext;
ze_result_t = zesDeviceGetEccState(device, &getState);
